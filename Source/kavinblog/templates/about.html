About Friend File



Friend File is a social blogging platform developed by Kavin Jindal. People can register on the website and write blogs on topics related to technology, coding, movies, life, wellness etc. People can come on the website and promote their content and creations here.The site's first version has been released and is still being worked on. The developer is working on adding more updates and features to enhance the user experience on Friend File.


Development Info


The site's frontend has been coded in HTML and styled using Bootstrap
The backend part has been coded in Python using Django.
